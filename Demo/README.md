# MounthAnalytic Demo

Це демо-пакет MounthAnalytic з анонімізованими даними.

## Що анонімізовано

- Реальні акаунти дизайнерів і баєрів
- Notion links
- Media/file paths
- Task IDs
- Тексти задач і Post
- GEO та мови

Числові значення, статуси, дати, пріоритети, типи креативів і кількість креативів залишені, щоб аналітика виглядала реалістично.

## Запуск

```bash
cd Demo
python3 -m http.server 8090
```

Потім відкрити:

```text
http://localhost:8090
```
